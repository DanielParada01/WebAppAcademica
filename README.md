# webappacademica

Es una pagina web dedicada a la creación y búsqueda de registro de alumnos docentes materias utilizando llaves primarias y foráneas, creación de Base De Datos independiente para hacerle llamado de los registros a la página web vistos en las clases de programacion 1.

Elaborada con #visual estudio 2022, utilizando #Web Api, y framework #Bootstrap, código utilizado de las practicas de la clase de programación 1, sin embargo contiene problemas de conexión a la base de datos, los datos han sido cargados las tablas han sido creadas y la base de datos existe sin embargo la conexión no tiene éxito al 100%, por otra parte, todo el código funcionando animaciones, tablas, opciones, todos los archivos HTML y Js funcionando.
